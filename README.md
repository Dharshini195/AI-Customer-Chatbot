🤖 AI Customer Support Agent

An intelligent customer support assistant built using FastAPI + LangChain + Groq LLM + Streamlit, capable of:

Handling customer queries conversationally
Performing real actions (order status, cancellation, ticket creation)
Using company policy knowledge (RAG)
Maintaining session-based memory
Following strict decision logic like a real support agent
🚀 System Overview

This project is a tool-using AI agent that behaves like a real customer support representative.

Core Capabilities:
🧠 Understands user queries using LLM
🔧 Executes backend tools (order status, cancel, ticket)
📚 Retrieves company policies (RAG)
💬 Maintains conversation memory
🎯 Follows strict decision rules via prompt engineering
